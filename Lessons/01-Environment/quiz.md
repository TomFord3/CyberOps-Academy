# Lesson 1 Quiz

1. What does git add do?
- Git add adds files to a staging area that can be included in the next commit.
2. What is a commit?
-  Commit creates a snapshot of all staged files. This is a comment message of what was changed since that last update.
3. What does git status show?
-  Git status shows the current state of the github repository.
4. Why doesn't Git track empty folders?
- Git tracks files, not folders. Since an empty folder contains no files, Git has nothing to save or track.
5. Explain the Git workflow in your own words.
- User create a file, User adds information to that file, User saves the file, User runs git status to see if the file is or is not being tracked User runs git add "FileName" to add either a specific file or git add . to add all the files to a tracked list. Then user runs git commit -m to add their comments regarding the changes Once the file is added, user then can run git push to finalize the commit.
