## Varaiable
- a named storage location used to hold data that can change during the execution of a program.

        ex: number = "1" or name = "Jim"

## fstring
- a formatted string literal introduced in Python 3.6 that provides an efficient and readable way to embed variables or expressions directly into strings using curly braces

       ex: print(f"name: {name}")

## Input  
- collect user input from the keyboard

        ex: input("What is your name: ")

## Line Breaks
- used to add space inbetween lines of code

        ex: "print()" or "\n"
